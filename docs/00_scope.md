# Stage 0 — MVP scope (зафиксировано)

## Что делаем в MVP
### Доступ и аккаунт
- Регистрация/логин: **email + пароль**.
- Доступ к базе и функциям — только авторизованным.
- Гостям показываем экран/баннер: «нужна регистрация».

### Профиль
- Публичные поля (видны до матча): бренд, описание, регион, маркетплейсы, категории, логотип.
- Скрытые поля (видны **только после матча**): **юридическое название, ИНН, телефон RU, контакты**.
- Бейдж доверия: базовый скоринг заполненности.

### Проекты (до 3)
- У пользователя может быть до **3 проектов**.
- Проект можно создать в ЛК или при отправке заявки.
- Если проектов уже 3 — предлагаем **удалить или заменить**.

### Карточки/объявления (3 типа)
- Типы: `collab` (партнёрство), `service` (услуга), `supplier` (поставщик).
- Создание, редактирование.
- Публикация/снятие с публикации (черновики не видны другим).
- Поиск: по строке + фильтры.

### Матчинг
- Заявка на match всегда сопровождается выбранным проектом + сообщением.
- Двустороннее подтверждение:
  - Получатель принимает (шаг 1)
  - Инициатор подтверждает (шаг 2)
  - После этого создаётся Match и раскрываются скрытые поля (всё сразу)
- Разорвать сотрудничество (terminate/unmatch) — обязательно. После разрыва скрытые поля снова скрываются.

## Что исключаем из MVP
- Подписки/оплата/премиум.
- Встроенный чат/звонки.
- Модерация.
- Автоматическая проверка ИНН/судебки по внешним сервисам.
- Рейтинги/отзывы.
- Интеграции с WB/Ozon.
